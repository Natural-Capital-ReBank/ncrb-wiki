# Quickstart Guides

Get set up and ready to use NCRB in a few steps.

---

## Recommended Order

| Step | Guide | What You'll Do |
|---|---|---|
| 1 | [Install MetaMask](./metamask) | Install the MetaMask browser extension and create your wallet |
| 2 | [Add Avalanche Networks](./avalanche-network) | Add Avalanche Fuji testnet and C-Chain mainnet to MetaMask |
| 3 | [Add XRPL EVM Networks](./xrpl-network) | Add XRPL EVM testnet and mainnet to MetaMask |
| 4 | [Add USDC to Your Wallet](./add-usdc) | Import the USDC token on each network so you can buy credits |
